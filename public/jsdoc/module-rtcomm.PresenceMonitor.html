Class: PresenceMonitor

rtcomm. PresenceMonitor

new PresenceMonitor()

This object can only be created with the getPresenceMonitor function.

The PresenceMonitor object provides an interface for the UI Developer to monitor presence of other EndpointProviders that have published their presence w/ the publishPresence function.

Once created, it is necessary to 'add' a topic to monitor. This topic can be nested and will look something like: 'us/agents' in order to monitor the presence of agents in the US.

This can go as deep as necessary.

The presenceData is kept up to date in the PresenceMonitor.getPresenceData() object.

Example
// After creating and initializing the EndpointProvider (EP)

var presenceMonitor = EP.getPresenceMonitor();
presenceMonitor.add('us/agents');
var presenceData = presenceMonitor.getPresenceData();
